## Introduction

This is an AVRPascal based replacement for the ACCII climate control system. It is under heavy development and is currently in the prototype phase. 

A FORTH and Pascal version are inluded. The FORTH version will be used for hardware setup and validation and the Pascal version will be used in production as the final firmware.

## Planned Features

```
All factory vent modes
Automatic mode
Blower control
Intellegent defrost
```

## Hardware

```
Arduino Nano
12v Buck Converter
4x 12v Vac Solenoids
MOSFETs
Buttons
LEDs
2x Analog temp sensors
```
